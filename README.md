# Sahl AI — Arabic ASR System for Medical Documentation

> Speech-to-text system built for Saudi Arabic dialect clinical documentation.

🔗 [sahl.ai](https://sahl.ai)

## Overview

Sahl AI is a specialized Arabic speech recognition system built for medical documentation in Saudi healthcare settings. The platform allows doctors and clinicians to dictate notes naturally in Saudi Arabic dialect and receive structured, accurate clinical transcriptions in real time, removing the need for manual documentation entirely. Built with advanced ASR technology and domain-specific LLM fine-tuning on Arabic medical terminology, Sahl AI integrates directly into existing clinical workflows.

## Problem

Doctors in Saudi healthcare settings were spending significant time on manual clinical note-taking after every patient consultation. Standard speech-to-text tools did not support Saudi Arabic dialect and lacked the specialized medical vocabulary needed for clinical accuracy. Generic ASR models produced frequent errors on medical terminology, diagnoses, and Arabic dialect speech patterns, making them unusable in clinical environments and creating a major administrative burden that reduced time doctors could spend with patients.

## Solution

Built a specialized Saudi Arabic ASR system fine-tuned on medical domain vocabulary and Saudi dialect speech patterns. Applied LLM fine-tuning techniques to train the model on clinical terminology, prescription language, and diagnostic phrasing specific to Arabic-speaking healthcare contexts. Developed a structured output system that formats transcriptions into organized clinical note sections automatically, and integrated the platform directly into clinic and hospital workflows for seamless dictation and storage within existing medical record systems.

## Responsibilities

- Designed the ASR architecture for Saudi Arabic dialect speech recognition
- Applied domain-specific LLM fine-tuning on Arabic medical terminology and clinical language
- Built the structured output engine to format raw transcriptions into organized clinical notes
- Integrated the speech-to-text system with existing clinical workflow and record management systems
- Optimized model performance for real-time transcription accuracy in a medical production environment
